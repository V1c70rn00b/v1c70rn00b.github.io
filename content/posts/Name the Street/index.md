# Name That Street

Challenge description

```jsx
There are plenty of areas to walk in cities around the world, and this street is one of them.

But where is it?

What is the name of this street?
```

The image can be accessed from this [link](https://challenge.bellingcat.com/assets/street-BAy8BoR_.jpg)

We shall start by first analyzing the image as shown below.

![image.png](image.png)

From the image above, we can check the number plate is of which country and we have a store name of JP Stoffe. 

We can tackle this in two ways, one by searching for the store to see which countries have it and two, by searching the number plate to find it is based to which country.

From this [site](http://www.worldlicenseplates.com/), the plate is a German plate. We can now search for the store to find it is in which part of Germany .

Searching on google maps, I found the following three places onn the map.

![image.png](image%201.png)

After going through the three places, the JP Stoffe at Frankfurt,provided as with the answer to our question and found the street.

![Screenshot 2026-03-11 112339.png](Screenshot_2026-03-11_112339.png)

Answer: **`Stiftstraße`**
